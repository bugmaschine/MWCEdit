# MWCEdit

**MWCEdit** is a modern, crossplatform save game editor for **My Summer Car** and **My Winter Car**.

It has a simple goal: make save editing safe, clear, and easy to use, without exposing raw values or risking save corruption.

Join the [Discord](https://discord.gg/yewzNFdkSr) to make suggestions and for help

---

## Features

- Automatic game detection  
  Detects whether the save belongs to MSC or MWC and adjusts available features accordingly.
  
- Safe in-place save editing  
  Easily edit things without having to dive in deep into editing.
  
- Cross platform support  
  Native Windows and Linux support
  
- Update checking
